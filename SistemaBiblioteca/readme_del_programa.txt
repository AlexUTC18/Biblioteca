Pasos para la instalacion y ejecucion del proyecto
Sistema de Gestion de Prestamo Bibliotecario

Paso 1
Tener instalado 
-MYSQL
-Editor de PHP

Paso 2
- Descargar la Carpeta "Sistema Biblioteca" en la Carpeta "HTDOCS" 
- Levantar los servicios de Xammp 
- Importar la BDD, "Biblioteca" al Gestor de Mysql

Paso 3
- Si todo se hizo de forma correcta En el navegador Buscar el proyecto: 
- http://localhost/sistemabiblioteca/cpanel
